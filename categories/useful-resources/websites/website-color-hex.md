## Описание / Description

ColorHexa - это мощный онлайн-инструмент для работы с цветами, который предоставляет исчерпывающую информацию о любом цвете и помогает создавать цветовые схемы. Сайт позволяет конвертировать цвета между различными форматами, находить похожие цвета и генерировать палитры.

ColorHexa is a powerful online color tool that provides comprehensive information about any color and helps create color schemes. The site allows converting colors between different formats, finding similar colors, and generating palettes.

## Основные возможности / Key Features

- **Конвертация цветов** между различными форматами (HEX, RGB, HSL, CMYK и др.)
- **Детальная информация** о каждом цвете (комplementary цвета, оттенки, тени)
- **Генератор палитр** на основе одного цвета
- **Градиенты** и переходы между цветами
- **Поиск похожих цветов** с точностью до 1%
- **Color conversion** between various formats (HEX, RGB, HSL, CMYK, etc.)
- **Detailed information** about each color (complementary colors, shades, tints)
- **Palette generator** based on a single color
- **Gradients** and transitions between colors
- **Similar color search** with 1% precision

## Как использовать / How to Use

1. Введите цветовой код в поисковую строку (HEX, RGB, название цвета)
2. Изучите детальную информацию о цвете
3. Используйте генератор палитр для создания цветовых схем
4. Экспортируйте результаты в нужном формате

1. Enter color code in the search bar (HEX, RGB, color name)
2. Study detailed information about the color
3. Use palette generator to create color schemes
4. Export results in the desired format
