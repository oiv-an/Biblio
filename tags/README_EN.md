The tag system allows finding related materials between different categories and provides flexible navigation through the library.

## How Tags Work

Each material in the library contains metadata with tags that help classify content and find related materials. Tags are used for:

- Content classification by topics
- Finding related materials between categories
- Filtering by languages (ru, en)
- Indicating material difficulty (beginner, intermediate, advanced)

## Main Tags

### Language Tags
- `ru` - materials in Russian
- `en` - materials in English
- `both` - bilingual materials

### Difficulty Tags
- `beginner` - for beginners
- `intermediate` - intermediate level
- `advanced` - for advanced users

### Thematic Tags
- `ai` - artificial intelligence
- `prompt` - AI prompts
- `code` - programming
- `design` - design
- `tool` - tools
- `website` - websites
- `tutorial` - educational materials
- `article` - articles
- `productivity` - productivity

## Adding New Tags

When adding new material:

1. Use existing tags when possible
2. Create new tags only when necessary
3. Follow tag naming rules:
   - Use lowercase letters
   - Don't use spaces (use hyphens)
   - Be concise
